CUDA-accelerated Monte Carlo for Exotic Option Pricing
PeriodMay 2025 AffiliationFlorida State University Associated withCourse Project · High-Performance Computing / Computational Finance RoleIndividual project
GPU computing Monte Carlo C++ / CUDA Derivatives pricing

Overview

Implemented Monte Carlo pricing for exotic options with a GPU-accelerated pipeline in CUDA, focusing on throughput, variance reduction, and reproducibility. Benchmarked against optimized CPU baselines and profiled performance bottlenecks at the kernel and memory levels.

GitHub Write-up (PDF) Slides Code

Contributions

  • Designed a GPU Monte Carlo engine for pricing path-dependent payoffs with parallel simulation and reduction.
  • Implemented variance reduction (e.g., antithetic variates / control variates where appropriate) and measured impact on error vs runtime.
  • Optimized kernel launches, memory access patterns, and RNG strategy; validated numerical correctness vs CPU reference.

Methods & technical notes

  • Models: Black–Scholes baseline; extensions depending on payoff structure.
  • Numerics: PRNG / normal sampling; payoff evaluation; confidence intervals; convergence diagnostics.
  • Performance: occupancy, memory bandwidth, and end-to-end speedups; profiling with Nsight (or equivalent).

Results

  • Achieved significant speedup over CPU baseline on large path counts; reported throughput (paths/sec) and error bars.
  • Demonstrated variance reduction improvements at fixed runtime for select payoffs.
