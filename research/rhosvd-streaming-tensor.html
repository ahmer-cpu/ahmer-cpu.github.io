Warm-started higher order randomized SVD (rHOSVD) on GPUs for streaming tensor data
StatusOngoing research RoleResearch assistant SupervisorDr. Kai Zhao DepartmentDepartment of Computer Science, Florida State University
GitHub Presentation

Overview

This project develops warm-started randomized higher-order SVD algorithms optimized for GPUs to efficiently process streaming tensor data. The goal is to maintain low-rank Tucker decompositions of high-dimensional tensors with minimal latency and memory overhead, leveraging prior solutions to accelerate convergence on new data arrivals.

Progress
