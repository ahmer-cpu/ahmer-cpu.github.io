Overview

Developed a reusable library of numerical analysis and Monte Carlo algorithms with emphasis on stability, convergence, and reproducibility. Implemented in MATLAB and Julia to facilitate comparison and interoperability.

GitHub

Contributions

  • Implemented classical numerical solvers: ODE/PDE, linear systems, root-finding, integration.
  • Developed Monte Carlo samplers: rejection, importance sampling, MCMC (Metropolis–Hastings, Gibbs).
  • Added comprehensive tests, documentation, and performance benchmarks; enforced code review standards.

Methods & technical notes

  • Numerical: RK4, implicit Euler, finite differences, sparse solvers.
  • Stochastic: Sobol sequences, Mersenne Twister, adaptive MCMC diagnostics.
  • QA: Convergence tests, error estimates, reproducibility checks (seeding, precision).

Results

  • Library integrated into course coursework and personal research projects.
  • Benchmark comparisons validate against established libraries (SciPy, StatsModels).

Notes

Open source under MIT license. Dual MATLAB/Julia implementation promotes language agility and portable knowledge.
