Equities Portfolio Optimization using James–Stein Shrinkage
Period2024 AffiliationFlorida State University Associated withCourse Project · Quantitative Portfolio Management RoleIndividual project StatusCompleted
Portfolio optimization Covariance estimation Shrinkage methods Backtesting

Overview

Studied covariance shrinkage techniques for mean–variance portfolio optimization and evaluated their out-of-sample risk–return performance. Compared James–Stein shrinkage with Ledoit–Wolf and sample covariance on real equity returns.

GitHub Write-up (PDF)

Contributions

  • Implemented James–Stein, Ledoit–Wolf, and sample covariance estimators; applied to mean–variance optimization.
  • Conducted out-of-sample evaluation using rolling windows; measured Sharpe ratio, turnover, and tracking error.
  • Analyzed shrinkage intensity vs estimation error trade-offs; documented finite-sample behavior.

Methods & technical notes

  • Estimators: James–Stein shrinkage toward identity; Ledoit–Wolf shrinkage toward constant-correlation target.
  • Data: S&P 500 constituents; monthly returns over multiple holding periods.
  • Evaluation: Sharpe ratio, Calmar ratio, portfolio weights stability, transaction costs.

Results

  • James–Stein shrinkage improved out-of-sample Sharpe ratio by ~5–12% vs sample covariance.
  • Optimal shrinkage intensity identified via cross-validation; robustness verified on subsamples.

Notes

Implementation includes numerical stability checks. Math: shrinkage is $$\hat{\Sigma} = \alpha \Sigma_{\text{sample}} + (1-\alpha) \Sigma_{\text{target}}$$ where $\alpha$ minimizes prediction error.
